# 📊 Sales Data Analysis Project

## 🧠 Overview
This project analyzes sales data using Python to extract business insights.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Analysis
- Monthly Sales Trends
- Top Selling Products
- Category Performance
- Regional Sales Distribution

## 📈 Key Insights
- Electronics is the top-performing category
- North region has highest sales
- Laptop is the best-selling product

## 🚀 Purpose
This project demonstrates Data Analysis skills for entry-level Data Analyst roles.
